## 还可以优化的地方

- 优化做成3级缓存，在内存里程序一关闭就没有了，但是我们之前查过的数据应该想办法再做一级缓存，用 redis
- 数据库从一读多写改为多读多写
- 连接不同的db，分多个库去搞
    - 这就已经是分布式了
    - 比如我们按省份和年龄段都划分过了，我们可以按照区域（华北，东北等等），划分成6个分布式的库。然后当需要搜索某人的时候，对6个db，同时查6个库的结果，汇总起来，再返回
